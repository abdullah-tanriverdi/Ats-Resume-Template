# ATS Resume Template
ATS uyumlu, HTML ve CSS tabanlı bir CV (özgeçmiş) şablonu.

## Amaç

Bu proje, adayların CV’lerinin Applicant Tracking System (ATS) filtrelerinden geçme olasılığını artırmak amacıyla oluşturulmuştur.  
Minimal, semantik ve metin odaklı bir yapı kullanılarak otomatik tarama sistemleriyle maksimum uyumluluk hedeflenmiştir.

## Proje Hakkında

Bu şablon; sade, tek kolonlu ve Applicant Tracking System (ATS) sistemleriyle uyumlu olacak şekilde tasarlanmıştır.  
Teknik pozisyonlara başvuran adaylar için optimize edilmiştir.  
Semantik HTML ve CSS yapısı sayesinde ATS sistemleri tarafından doğru şekilde ayrıştırılabilir.

## Nasıl Kullanılır?

1. Repository’yi klonlayın veya indirin.
2. `index.html` dosyasını açın.
3. İçerikleri kendi bilgilerinizle değiştirin.
4. PDF çıktısı almak için:
   - Tarayıcıda `Ctrl + P`
   - "PDF olarak kaydet" seçeneğini kullanın.
